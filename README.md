Student Management System

Overview
Student Management System is a console-based Java application used to manage student records efficiently.
The project allows users to add, view, update, search, and delete student information using a menu-driven interface.

This project is built using Java concepts such as:

Classes and Objects
ArrayList
Methods
Loops and Conditions
OOP Concepts
Input Validation

Features
Add new student records
View all students
Update student details
Delete student records
Search student by ID
Menu-driven console interface
Simple input validation
Formatted console output

Technologies Used
Java
ArrayList
Scanner Class
OOP Concepts

Project Structure
StudentManagementSystem.java

How to Run
Step 1: Compile the Program
javac StudentManagementSystem.java
Step 2: Run the Program
java StudentManagementSystem
